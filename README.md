# Employee Management System

A Java-based console application to manage employee records using CRUD operations, collections, and file handling.

# Features
- Add, view, search, and delete employees
- Uses ArrayList and HashMap
- File persistence using serialization
- Salary and department reports
- Exception handling

# Technologies Used
- Java
- OOP Concepts
- Collections Framework
- File Handling

# Project Structure
EmployeeManagementSystem/
├── src/
├── data/
├── reports/
├── docs/
└── README.md

# How to Run
cd src
javac .\*.java
java EmployeeManagementSystem
